# Course Registration System

## Overview
The **Course Registration System** is a project developed for the *System Development and Technology* course.  
This system allows students to register for courses, lecturers to manage course applications, and administrators to maintain course and lecturer information.

The system is designed with **three main user roles**, each with different functionalities.

---

## User Roles and Features

### 1. Student
Students can manage their course registration through the system.

**Functions:**
- Register for courses
- Update registered courses
- View registered courses

---

### 2. Lecturer
Lecturers manage and review course applications submitted by students.

**Functions:**
- Approve course applications
- View list of courses
- View list of registered students

---

### 3. Admin
Administrators maintain the overall course and lecturer information in the system.

**Functions:**
- Manage courses
- Manage lecturers
- Register new lecturers

---

## Purpose of the Project
This project was developed to demonstrate concepts learned in the **System Development and Technology** course, including:
- System design
- Role-based access control
- Basic course management workflow
